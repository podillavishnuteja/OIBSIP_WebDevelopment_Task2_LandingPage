# OIBSIP_WebDevelopment_Task1_Landingpage

## Project Title
Landing Page

## Objective
The objective of this project is to create a Landing page is to create a visually appealing, user-friendly digital presence for an authentic Indian restaurant.

## Features
-Hero Section

-About Section

-Menu Section

-Gallery Section

-Navigation & Header

-Footer

-Design & UX Enhancements

-Responsive Design

## Project Overview
Savory Haven is a premium restaurant website template designed for restaurants aiming to highlight

-Signature dishes

-Ambiance and interior

-Heritage and culinary philosophy

-Contact and reservation information

**The design focuses on**

-Elegant typography using Google Fonts (Playfair Display and Outfit)

-A warm, sophisticated color palette

-Smooth transitions and hover effects

-Full responsiveness across devices

## Technologies Used

-HTML5 – Semantic markup for structure.

-CSS3 – Styling with variables, transitions, and grid/flexbox layouts.

-Google Fonts – Typography: Outfit and Playfair Display.

-Responsive Design – Media queries for multi-device support.

-Images – High-quality stock images via Unsplash and authentic food photography.

## Customization

-Colors: Modify CSS variables in :root for primary, accent, and background colors.

-Fonts: Adjust Google Fonts in <head> if needed.

-Images: Replace image URLs in index.html for hero, about, menu, and gallery sections.

-Menu Items: Duplicate .menu-card blocks in the menu section to add more dishes.

## Responsive Design
The website layout adapts to different screen sizes:

-Desktop: Full grid layout for menu and gallery.

-Tablet (≤1024px): 2-column grids for gallery and footer.

-Mobile (≤768px): Single-column layout, hidden navigation replaced with custom mobile menu (can be added).
